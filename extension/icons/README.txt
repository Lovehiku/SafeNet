ICON FILES REQUIRED
===================

This directory should contain three PNG icon files:

1. icon16.png  (16x16 pixels)
2. icon48.png  (48x48 pixels)
3. icon128.png (128x128 pixels)

To generate these icons:
- Open create-icons.html in your browser
- Click "Generate Icons"
- Right-click each canvas and save as the corresponding filename

Or use any image editor to create shield/security-themed icons.
